# Inox Pharma - Gestion des Ventes Éclatées

Application web de suivi et d'analyse des ventes par pharmacie pour la société Inox Pharma.

## Fonctionnalités
- Import des données de ventes éclatées (Excel)
- Tableau de bord avec graphiques
- Filtres par produit, client, période, agence
- Export des données

## Technologies
- PHP 8+
- MySQL
- Bootstrap 5
- Chart.js

## Installation
1. Cloner le dépôt
2. Importer le fichier `install/create_tables.sql` dans MySQL
3. Configurer `config/database.php`
4. Lancer `php -S localhost:8000`
